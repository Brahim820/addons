Restful API Documentation

Getting Started

  1. User requests for access-token with username and password
  2. Odoo will validate the credentials
  3. Odoo provides a signed access-token to the client
  4. Client/client stores that token and sends it along with every request
  5. Server verifies token and responds with data
    6. import requests headers = {'content-type': 'application/x-www-form-urlencoded', 'charset':'utf-8'} login, password = 'demo', 'password' # please replace these dummy value with login credentials database = 'demo' # please replace with the database name you want to connect to. base_url = 'https://www.examle.com'

How to request for access-token

request: data = { 'login': login, 'password': password, 'db': database } url = "{}/api/auth/token".format(base_url) requests.get(url=url, data=data, headers=headers)
response: { "company_id": 1, "user_context": { "lang": "en_US", "tz": "False", "uid": 5 }, "access_token": "b2ZpWcqjRLoPApgzHPRwdsisWVn1wl", "expires_in": "600", "uid": 5 }
